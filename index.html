Code

Get started with ScribbleFX in seconds. Copy, paste, and watch the magic happen.

Basic Usage

// Import ScribbleFX
import ScribbleFX from 'scribblefx' ;
// Add underline to any element
ScribbleFX . underline ( '.title' );

Advanced Options

ScribbleFX . underline ( '.heading' , {
style : 'scribble' ,
animation : 'draw' ,
duration : 2.0 ,
color : '#f97316' ,
trigger : 'scroll'
});

HTML Setup

<!-- Your HTML -->
<h1 class = "title" >
Amazing Headlines
</h1>
<!-- Include GSAP -->
<script src = "gsap.min.js" ></script>
<!-- Include ScribbleFX -->
<script src = "scribblefx.min.js" ></script>

Multiple Elements

// Apply to multiple elements
ScribbleFX . underline ( '.highlight' , {
style : 'wave' ,
stagger : 0.2
});
// Chain different styles
ScribbleFX . underline ( '.cta' )
. then (() => console . log ( 'Done!' ));
